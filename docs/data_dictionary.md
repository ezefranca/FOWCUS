# Data Dictionary

All derived tables are generated by `scripts/build_dataset.py` from the raw FOWCUS workbook. CSV files live in `data/processed/csv/`; Parquet files live in `data/processed/parquet/`.

## `fowcus_parts`

One row per commodity part.

| Field | Type | Description |
| --- | --- | --- |
| `record_id` | string | Stable row identifier assigned in source workbook order. |
| `commodity_id` | string | Stable commodity identifier derived from food group, source item code, item label, and a short source-key hash. |
| `food_group` | string | Canonical food-group sheet name. |
| `source_group` | string | Original `group` value from the workbook row. |
| `item_code` | string | Source commodity code exactly normalized as text; `aggregation` identifies grouped seafood categories. |
| `item_code_kind` | string | `commodity_code` or `aggregation`. |
| `item` | string | Source commodity item label. |
| `part` | string | Commodity component or by-product label. |
| `mass_fraction` | number | Fraction of the commodity mass represented by the part. |
| `mass_percent` | number | `mass_fraction * 100`. |
| `accounted_in_fao` | nullable boolean | Whether the source workbook says the part is included in FAO/FAOSTAT production accounting. Blank means the source workbook did not provide a value. |
| `avoidability` | string | Normalized category: `avoidable`, `potentially_avoidable`, or `unavoidable`. |
| `reference_id` | string | Key into `fowcus_references`. |
| `notes` | string | Source notes, if present. |
| `data_manipulation` | string | Source description of how the value was used or derived, if present. |
| `source_sheet` | string | Workbook sheet name. |
| `source_row_number` | integer | Excel row number in the upstream workbook. |

## `fowcus_commodities`

One row per commodity-code/item combination.

| Field | Type | Description |
| --- | --- | --- |
| `commodity_id` | string | Stable commodity identifier used by `fowcus_parts`. |
| `food_group` | string | Canonical food group. |
| `item_code` | string | Source commodity code. |
| `item_code_kind` | string | `commodity_code` or `aggregation`. |
| `item` | string | Source commodity item label. |
| `n_parts` | integer | Number of part rows linked to the commodity. |
| `mass_fraction_sum` | number | Sum of `mass_fraction` across linked parts. |
| `mass_balance_error` | number | `mass_fraction_sum - 1`. |
| `mass_balance_status` | string | `pass` when the sum equals 1 within floating-point tolerance; otherwise `review`. |
| `accounted_mass_fraction_sum` | number | Sum of parts where `accounted_in_fao` is true. |
| `non_fao_mass_fraction_sum` | number | Sum of parts where `accounted_in_fao` is false. |
| `unknown_fao_accounting_mass_fraction_sum` | number | Sum of parts where `accounted_in_fao` is blank in the source. |
| `avoidable_mass_fraction` | number | Sum of avoidable parts. |
| `potentially_avoidable_mass_fraction` | number | Sum of potentially avoidable parts. |
| `unavoidable_mass_fraction` | number | Sum of unavoidable parts. |
| `source_sheets` | string | Source sheet names contributing to the commodity. |

## `fowcus_references`

Deduplicated source references.

| Field | Type | Description |
| --- | --- | --- |
| `reference_id` | string | Stable hash identifier for the reference text and URL pair. |
| `reference_text` | string | Source citation text from the workbook. |
| `reference_url` | string | Source URL or DOI where present. |
| `n_records` | integer | Number of part records linked to the reference. |

## `fowcus_classification_notes`

Normalized rows from the workbook classification notes sheet.

| Field | Type | Description |
| --- | --- | --- |
| `classification_note_id` | string | Stable row identifier. |
| `food_group` | string | Food group after fill-down normalization. |
| `classification_note` | string | Classification note after fill-down normalization. |
| `item` | string | Item covered by the note. |

## `fowcus_quality_flags`

Generated review table for source-data conditions that should be visible to analysts.

| Field | Type | Description |
| --- | --- | --- |
| `flag_id` | string | Stable quality-flag identifier. |
| `scope` | string | `commodity` or `part`. |
| `severity` | string | `info` or `warning`. |
| `check` | string | Name of the check that produced the flag. |
| `record_id` | string | Part row identifier when the flag applies to a row. |
| `commodity_id` | string | Commodity identifier when applicable. |
| `message` | string | Human-readable issue description. |
